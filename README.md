# New ReportNG
对reportng做了二次开发，增加了以下新功能:
1. 首页添加汇总图表
2. 截图已弹窗方式展示，使多图情况下展示效果更加美观易读
3. 增加图片轮播展示形式，使较大图片如PC端截图，展示更加美观
